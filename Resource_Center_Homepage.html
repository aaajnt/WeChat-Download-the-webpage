    # 开放平台支持接入如下类型的第三方应用：

    # 1.移动应用

    • 移动应用的接入指引以及开发优势等，请查看https://open.weixin.qq.com/cgi-bin/frame?t=home/app_tmpl&lang=zh_CN
    • 移动应用的详细开发指引，可查看https://developers.weixin.qq.com/doc/oplatform/Mobile_App/Access_Guide/iOS.html

    # 2.网站应用

    • 网页应用的接入指引以及开发优势等，请查看https://open.weixin.qq.com/cgi-bin/frame?t=home/web_tmpl&lang=zh_CN
    • 网页应用的详细开发指引，可查看https://developers.weixin.qq.com/doc/oplatform/Website_App/WeChat_Login/Wechat_Login.html

    补充，WeDemo 开源项目（微信登录、微信分享等功能示例，包括客户端（iOS）及后台源代码，开发者可参考源代码完成开发，安全、便捷地在 App 中实现微信登录、分享等功能。查看WeDemo 开源项目 ）

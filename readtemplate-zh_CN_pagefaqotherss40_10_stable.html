微信1.0 for诺基亚S40全新发布

发布日期：2013-05-31

发布版本：微信 1.0 for S40

微信是一款免费的手机通信软件，支持通过手机网络发送语音短信、图片和文字，带给朋友们全新的移动沟通体验。支持iOS、Android、塞班、WP、诺基亚S40等多种平台手机，界面可显示简体中文、繁体中文、英文、西班牙语、印尼语等多种语言。

主要支持机型：
Nokia Asha 303 / 305 / 306 / 307 / 308 / 309 / 310 / 311
Nokia Asha 3030 / 3050 / 3060 / 3070 / 3080 / 3090 / 3100 / 3110
Nokia X3-02 / Nokia C3-01

1.0 版本主要特性如下：

- 语音短信，让聊天更简单。按住拨号键开始说话，松开拨号键发送语音消息

 

- 可发送文字、图片、表情消息

 

- 群聊功能，并可以保存群到通讯录中

 

- 从QQ好友和手机通讯录中，找到更多朋友

 

- 设置独立密码，登录更方便安全
